Why Commander

Commander is a mature administration panel coded and designed by nana_kon, with designs and logic brought from the well-praised CanaryAdmin administration panel. When you are using Commander, you will be amazed by the gorgeous user interface and the design which makes you to moderate server efficiently.

Apart from that, Commander has built-in support for custom commands, meaning that you do not need to make changes to the UI (like other administrative panels). Just put the command package into the package folder, and we will handle the rest for you automatically.

Also, it is open-sourced under the MIT license!

There's more reasons why you should use Commander rather than others, such as:

  • getfenv/setfenv free, Luau optimisations will not be disabled!

  • Easy to use API, no need to recreate stuff for your command!

  • Easy to configure UI, as long as you know how to code!

  • Everything is secured, no need to worry about exploiters!

  • 4 easy steps to set-up, you do not need to be a programmer to set the system up!

  • Extremely modular, UI and packages are done in a modular way!

Designed for human

We believe a good design is when it can serve the human effectively, so they can work on stuff efficiently, this is why we don't focus on making stuff that only looks cool, we focus on improving the efficiency.
